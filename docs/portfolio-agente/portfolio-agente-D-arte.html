01
Respiração Editorial
Evolução segura — hierarquia e whitespace intencional
Evolução

InspiraçãoVogue Brasil impresso — index mark gigante, grid modular com gap generoso, tipografia em escala progressiva Cormorant + Lato. O "01" em 9rem é a marca d'água que ancora a composição sem decoração gratuita.

Problema resolvidoGap zero eliminado (24px entre cards). Hierarquia tipográfica: nome em Lato 900 uppercase, meta em muted 72%, CTA reveal no hover. A linha olive cresce da base do card indicando interatividade com elegância.

Trade-offMais vertical que o layout atual. Em telas 1280px o grid de 5 colunas fica compacto — cards finos, mas a proporção 3:4 mantém o peso da imagem.

02
Polaroid de Película
Evolução segura — moldura analógica, rotação leve
Evolução

InspiraçãoLivros de fotografia analógica e moodboards editoriais — borda branca larga como moldura polaroid. A "fita adesiva" em creme no topo de cada card é o flair autoral: simples, referencial, sem exagero.

Problema resolvidoImagens sem enquadramento → polaroid cria moldura intencional. Rotações ±2° dão vida sem perder profissionalismo. Hover normaliza e eleva o card, criando hierarquia clara de foco.

Trade-offNome em Cormorant itálico sob a foto é mais delicado que bold uppercase — sacrifica um pouco de legibilidade rápida em favor de sentimento artesanal. Adequado para o público da Lillia.

03
Galeria com Ritmo
Evolução segura — alturas alternadas, overlay no hover
Evolução

InspiraçãoGalerias físicas de fotografia — works em alturas variadas criam movimento sem caos. CTA no topo direito é posicionamento inteligente: quem vai ver o portfólio completo está decidido antes de ver os cards.

Problema resolvidoMonotonia visual do grid uniforme. Alturas 420/340/420/340/420 criam ritmo sinusoidal. Overlay dark no hover revela texto diretamente sobre a imagem, unindo contexto e imagem sem espaço extra.

Trade-offO offset vertical dos cards pares exige que o container tenha altura suficiente — pequeno risco de corte em viewports muito estreitas. Texto sobre imagem depende do contraste da foto real.

Magazine Overlay
Ousada — fundo escuro, texto sobre imagem, Cormorant itálico
Ousada

InspiraçãoHarper's Bazaar, Aperture magazine — fundo quase-preto que faz as imagens respirarem como luminárias em sala escura. O nome da cliente em Cormorant 1.3rem itálico tem peso editorial real. Gap 3px entre cards cria tensão de mosaico.

Problema resolvidoImagens sem enquadramento → imagem é o card inteiro. Hierarquia em camadas: de grayscale para color no hover, texto aparece em sequência (nome → estúdio → CTA), criando reveal cinematográfico.

Trade-offFundo escuro quebra a continuidade com o restante do site (branco/creme). Depende de fotos com boa qualidade para o overlay não escurecer demais. Risco calculado — o contraste é deliberado.

Lista Vertical Assimétrica
Radical — layout documental, numeração romana, linha pivot
Ousada

InspiraçãoLayouts de fotógrafos documentais (arquivos Magnum, livros Sebastião Salgado) — o grid de 5 colunas é desconstruído em lista. A linha olive na borda esquerda é a espinha dorsal visual. Numeração romana amplifica o caráter artesanal.

Problema resolvidoGrid horizontal convencional → hierarquia de lista vertical cria espaço para nome grande, categoria e localização legíveis. A thumbnail em strip 120px é suficiente para reconhecimento sem ser o foco.

Trade-offQuebra radicalmente o padrão de portfolio em grid. Pode ser percebido como "minimalista demais" para clientes que esperam ver fotos grandes. A aposta é que o refinamento compensa — adequado para brand premium.

Ranking — craft visual

1°
04 — Magazine Overlay
Maior coerência entre identidade e execução. O fundo escuro transforma imagens em objetos de luz — exatamente o que fotografia de retrato merece. Hover em três camadas (cor, texto, CTA) é o hover mais sofisticado das 5. O Cormorant itálico sobre o gradient cria tensão tipográfica real. Risco mais alto, recompensa mais alta.
2°
01 — Respiração Editorial
O "01" em Cormorant 9rem como marca d'água âncora é o flair mais elegante entre as evoluções seguras. Hierarquia tipográfica clara: Cormorant para display, Lato para dados. A linha olive que cresce da base do card é o hover mais honesto — escala o que a imagem promete. Fácil de aprovar com qualquer cliente.
3°
05 — Lista Vertical Assimétrica
O design mais corajoso conceitualmente. A lista documental com numeração romana e espinha olive tem identidade autoral forte. O risco é real: clientes de fotografia esperam imagens grandes. Mas para um brand premium que quer se diferenciar de "mais um portfolio em grid", esta proposta abre conversas importantes.
4°
03 — Galeria com Ritmo
Alturas alternadas resolvem a monotonia sem abandonar o grid. O overlay hover é eficiente. Perde pontos por ser uma solução que já aparece em muitos sites de fotografia — o ritmo sinusoidal é bom, mas não suficientemente distinto. O CTA no topo direito é uma decisão inteligente de UX que compensa.
5°
02 — Polaroid de Película
O conceito é delicioso e a fita adesiva em creme é o melhor detalhe autoral do lote. Mas as rotações criam tensão de alinhamento que pode incomodar em produção com imagens reais de tamanhos variados. A proposta precisa de mais controle sobre as fotos para funcionar bem — em placeholder funciona, mas é a mais dependente de conteúdo real.
