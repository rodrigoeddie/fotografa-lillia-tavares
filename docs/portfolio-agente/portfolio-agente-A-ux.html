Proposta 3 · Evolução Moderada

Grid Assimétrico com Destaque Editorial

Quebra a monotonia de 5 colunas iguais introduzindo um card featured (2×) e 4 cards menores. A hierarquia visual guia o olho naturalmente; o primeiro trabalho recebe mais atenção. O trade-off é que o layout favorece um trabalho sobre os demais, o que pode gerar expectativas do cliente sobre ordenação.

Evolução moderada

Proposta 5 · Extrapolação

Accordion Horizontal — Foco e Periferia

Interação baseada em expansão: colunas colapsadas mostram apenas o nome rotacionado; ao clicar/focar, o card expande revelando a foto completa e o CTA. Cria uma micro-narrativa de descoberta. Risco: a interação não é óbvia sem onboarding; usuários mobile precisam da versão vertical (implementada). Requer JS mínimo.

Extrapolação ousada
Estados: Default (colapsado) · Active/Expanded · Focus-visible · Hover (cursor pointer) · Loading (painel 3)

Ranking · Melhor → Pior implementação

1

P3 · Grid Assimétrico com Destaque Editorial

Resolve hierarquia, clickability e responsividade sem introduzir dependências ou padrões de interação desconhecidos. O card featured comunica imediatamente qual trabalho merece atenção; os cards menores funcionam como suporte sem pedir atenção em excesso. Implementação pure CSS com mínimo JS — compatível diretamente com o Nuxt/GSAP atual.

Melhor custo-benefício · Implementação 1–2 dias
2

P5 · Accordion Horizontal — Foco e Periferia

A interação de expansão é memorável e diferencia o site. Em desktop funciona impecavelmente; em mobile a versão vertical é igualmente elegante. O risco é a discoverability — usuários que não testam o clique nunca veem o conteúdo expandido. Um hint visual (cursor animado ou label "clique") mitiga o problema.

Alto impacto · Requer JS leve · Testar com usuários
3

P1 · Gap + Hover Reveal

A proposta mais conservadora e de menor risco. Resolve o problema mais crítico (gap) e adiciona hover com elegância. A desvantagem é que mantém a estrutura Swiper atual, que já tem problemas de paginação e acessibilidade documentados. É uma evolução, não uma solução.

Menor risco · Menor impacto · Implementação meio-dia
4

P4 · Filmstrip Editorial — Contact Sheet

Visualmente mais distintivo e com uma narrativa de marca clara (laboratório fotográfico). O fundo escuro exige decisão estratégica de branding — não pode ser implementado de forma incremental sem revisar o restante da home. Melhor como proposta para uma landing page ou seção "about" do que para a home principal.

Alta distinção · Alto risco de marca · Decisão de direção criativa
5

P2 · Scroll Horizontal com Cards Maiores

Funcional e mobile-friendly, mas perde a visão panorâmica dos 5 trabalhos em desktop — o que é exatamente o que a seção se propõe a fazer ("explore meus últimos trabalhos" implica ver o conjunto). O carrossel horizontal em desktop é um padrão controverso: esconde conteúdo sem indicar claramente que existe mais. Adequado se o portfólio crescer para 10+ itens.

Melhor para portfólios grandes · Menos eficaz para 5 itens
