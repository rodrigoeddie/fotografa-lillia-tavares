Proposta 02

Faixa Horizontal — Ritmo editorial, célula CTA em terracota

Proposta 03

Card Elevado — Calendário como âncora com shadow premium

Assimetria intencional: lado esquerdo é editorial (texto largo, respiro vertical, CTA destacado), lado direito é o calendário como "peça" isolada num card branco com shadow profunda. A data selecionada aparece num mini-display abaixo do grid, criando um feedback imediato e satisfatório antes de enviar. Funciona bem como referência de upgrade futuro com mais info (valor do ensaio, disponibilidade confirmada).

Proposta 04

Tira Editorial — Inversão total, display serif, assinatura autoral

O risco calculado desta proposta: o dark mode terracota com tipografia display italic em 62px cria um momento de assinatura dentro da página — o usuário para, lê, é seduzido. O calendário inverte (dark field, dias em creme), o CTA se torna uma pílula bege que é o único elemento claro no campo escuro, criando hierarquia por contraste total. Inspiração: editorial de moda, não dashboard de software.

Proposta 05

Instante — 2 etapas, calendário em foco, CTA como revelação

Experiência sequencial: a seção começa limpa, com o calendário como protagonista absoluto. Quando o usuário seleciona uma data, o painel de confirmação desce com uma animação spring — a data escolhida aparece como chip, o CTA ganha contexto real. Elimina a desordem visual de mostrar tudo junto. O ritmo é: ver trabalho → querer → escolher data → confirmar. Cada etapa tem um único foco.

Avaliação Final

Ranking estético — melhor ao mais experimental

Critérios: coesão com o ensaio terracota, hierarquia clara, respeito ao tom acolhedor da Lillia, craft nos detalhes, viabilidade de implementação no Nuxt existente.

1
Proposta 3 — Card Elevado
A assimetria texto + calendário-card resolve o problema original das 3 colunas iguais sem romper com o site — o calendário ganha protagonismo visual merecido, o display de data selecionada adiciona satisfação interativa que as outras propostas não têm, e o CTA tem força sem briga com os demais elementos.
2
Proposta 5 — Instante
A experiência sequencial é a mais coerente com o tom acolhedor e personalista da Lillia — sente como uma conversa, não um formulário. O risco é que usuários menos ativos percebam o CTA como oculto; mas quem chegou até aqui já está motivado.
3
Proposta 1 — Refinada
A correção mais limpa do problema atual — todas as proporções consertadas, hierarquia clara, menor risco de rejeição pelo cliente por ser evolutiva. Perfeita como ponto de partida para aprovação rápida.
4
Proposta 4 — Tira Editorial
Tecnicamente a proposta com maior ambição autoral e craft tipográfico, mas o dark mode terracota profundo é um salto grande de linguagem para o site atual — pode funcionar num rebranding completo, mas pede mais convencimento para uma seção isolada. O risco vale em contexto certo.
5
Proposta 2 — Faixa Horizontal
A célula CTA em terracota sólido é a ideia mais clara desta proposta, mas o grid de 3 células sem nenhuma sobreposição de camadas resulta num design competente sem surpresa. Funciona bem como fallback seguro para versão mobile-first ou landing pages stripped-down.
